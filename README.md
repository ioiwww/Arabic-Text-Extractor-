Arabic Text Extractor 

This prgrarom allaws you to easily  extract arabic text from a pdf and scanned docments with python 


How To run :

create a virtual Environment 
___
    python -m venv venv 
___

Activate the venv you just created 

___

    source path/venv/bin/activate 
___

Install requirements 

___
    pip install -r requirements.txt
___


copy the pdf you want tp extract from into the /pdfs dir 

run extract_arabic_pdf..py

results will be stored in the /output  dir
